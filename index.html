About Us

Built by Developers,
for Developers

Built as a passion project by an independent developer to prove that a code editor can be both visually stunning and highly responsive.

un&script was born from a desire for clean code and truly elegant tooling. The goal was simple: combine the power of a modern code editor with a uniquely beautiful glassmorphism interface.

As an independent developer, I understand what it means to want a tool that doesn't just work, but feels incredible to use. Every detail has been crafted with aesthetics and simplicity in mind.

This project is completely free and completely local. No telemetry, no forced cloud accounts, just you and your code.

Independent Project

Created by a solo developer focused on pushing the boundaries of UI design in desktop applications.

Fast & Lightweight

Built on top of Electron with minimal overhead, ensuring quick startup times and responsive typing.

Glassmorphism

A custom frameless window design that breaks away from traditional boring grey boxes.

Privacy First

Your code stays on your machine. No hidden tracking, no telemetry, no data collection.

Features

Why Choose un&script?

We put care into every feature so you can focus on what matters most — writing great code.

Fast Startup

Optimized core built on Electron ensures the editor opens quickly when you need it.

Glassmorphism UI

A truly unique, semi-transparent interface that adapts to your desktop background for a premium look.

Monaco Engine

Powered by the exact same world-class code editing engine used in VS Code, giving you reliable syntax highlighting.

Project Explorer

Built-in file explorer lets you open entire project directories, seamlessly navigate files, and manage your workspace.

Git Integration

View file modifications at a glance with built-in Git status colors right inside your project tree.

100% Local

No telemetry, no hidden requests, no tracking. Your code remains securely on your machine at all times.
